# Back-Mobile

Proyecto backend para aplicación móvil.

## Descripción

Este es un proyecto backend desarrollado con Node.js para soportar una aplicación móvil.

## Instalación

```bash
npm install
```

## Uso

```bash
npm start
```

## Desarrollo

Este proyecto utiliza Git Flow para el manejo de ramas.
